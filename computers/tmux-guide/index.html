Tmux guide

2017-02-05
  • linux
  • tmux
  • unfinished
  • why use tmux? What do I need a multiplexer for anyway?
    • Lets you create splits and new windows in your terminal.
    • creates persistent session for ssh
  • configure while reading
    • muxconf() { tmux "$@" && echo "$@" >> ~/.tmux.conf ; }
    • muxconf set prefix C-a
  • default keybinds
    • <prefix> % - split to the right
    • <prefix> " - split below
    • <prefix> c - make a new window
    • <prefix> ! - make the current pane a new window
  • suggested settings
    • bind @ command-prompt "joinp -s '%%'"
    • bind C command-prompt "new-window '%%'"
